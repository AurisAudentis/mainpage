This is an open-source project aimed at the integration of architecture and the digital world. It allows you to use microprocessor chips to create a 3D model of a self-built structure, and to connect sensors to create responsive architecture. The project is written in a myriad of languages, including Javascript/Node.js for the backend, c/c++ for the Nodes and Python/Grasshopper for the 3D model. There is also a p5.js model availiable.
